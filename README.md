# JustTalk
一个仿照qq的通讯软件
已实现功能有 文本信息互传 图片互传 登录 注册 头像上传 好友添加
登录界面的球使用了自定义view与属性动画结合
添加好友的界面用了共享动画和Morph动画 共享动画用于图标转换 Morph用于实现平滑曲线，参考了网上的教程，本来还有背景虚化，没做出来，
